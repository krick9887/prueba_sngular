# Rick and Morty Character Database (Prueba Sngular)

Esta es una prueba técnica desarrollada por Enrique Martínez García, que consiste en una aplicación web para explorar personajes del universo de Rick y Morty, consumiendo la [API de Rick and Morty](https://rickandmortyapi.com/).

## Demo en vivo

Puedes ver el proyecto desplegado aquí: [https://krick9887.github.io/prueba_sngular]
